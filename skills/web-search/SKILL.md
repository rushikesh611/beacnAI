# Web Search Skill

When the user asks about current events, news, recent developments, prices, scores,
or anything that may have changed, use the `web_search` tool.

## Guidelines
- Always cite the source URL in your reply
- Prefer recent results (sort by date when possible)
- Summarize findings rather than quoting verbatim
- If results are ambiguous, search again with a refined query
